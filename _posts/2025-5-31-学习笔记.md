有些韩语词发音和日文好相似，查了下词源都是中文，虽然意思有点不一样，但这样很好记

| 韩语                       | 日语                                                                      |
| ------------------------ | ----------------------------------------------------------------------- |
| 계산 gye san (calculation) | 決算 (settlement of accounts, especially in business/financial contexts ) |
| 수고 su go                 | 受苦(じゅく)                                                                 |
| 기사 gi sa（技士）             | 技士(ぎし)                                                                  |
| 학생 hag saeng(学生)         | 学生                                                                      |
| 학교 hag gyo (学校)          | 学校                                                                      |
| 선생 sonsae(先生)            | 先生                                                                      |
| 의사 wui sa (医者)           | 医者                                                                      |
| 가수 ka su (歌手)            | 歌手(かしゅ)                                                                 |
| 소주 so ju （酒)             | 焼酒(しょうしゅ)                                                               |
| 주문 ju mun（注文）            | 注文(ちゅうもん)                                                               |
| 회사원 hoe sa won （会社员）     | 会社員                                                                     |

## 容易忘记的语法

### 이에요 예요

- Use **이에요** after a consonant.
- Use **예요** after a vowel.

### 이 가

- **이** is used after a consonant
- **가** is used after a vowel

### 은 는

- 은 is used after a consonant
- 는 is used after a vowel
